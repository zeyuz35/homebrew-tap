# homebrew-mosek

## How do I install these formulae?

`brew install zeyuz35/mosek/<formula>`

Or `brew tap zeyuz35/mosek` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "zeyuz35/mosek"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
